You may add a business location with a straightforward user interface. This location allows you to get a link for directions that starts from where you are by sending an email/text, which is quite cool! This’ll include phone number, fax number, two-line address, city, state, zip code, and contact e-mail address. All you need to do after this is just copy and paste a simple short code (a small piece of code that embeds a more complex object) into any post/page. The locations is responsive to mobile and desktop browsers. Css is already included to make it a default to give a better looking layout for the maps. = Upgrades =The GoldPlugins can allows you to import your Locations from a CSV file by upgrading on this site: http://goldplugins.com/our-plugins/locations/upgrade-to-locations-pro/?utm_source=wp&utm_campaign=desc_upgrade1 "Upgrade to Locations Pro" or forLocations Pro: http://goldplugins.com/our-plugins/locations/upgrade-to-locations-pro/?utm_source=wp&utm_campaign=desc_upgrade2== How To Set Locations Up on Your Website? ==To Install the Plugin:1.	On Cpanel Dashboard, go to ‘public_html/wp-content/plugins’ directory. This is where you’ll need to upload the ‘locations’ file. 2.	On WordPress dashboard, go to ‘Plugins’ and activate the ‘locations’.3.	You’re good to go! Add a Location: When you are in the ‘locations’ page, click on ‘Add New Location’ from within WordPress. You may also edit/delete the location there as well.
